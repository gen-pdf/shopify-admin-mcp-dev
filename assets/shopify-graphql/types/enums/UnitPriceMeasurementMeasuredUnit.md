# Enum: `UnitPriceMeasurementMeasuredUnit`

The valid units of measurement for a unit price measurement.

## Values

- `ML` — 1000 milliliters equals 1 liter.
- `CL` — 100 centiliters equals 1 liter.
- `L` — Metric system unit of volume.
- `M3` — 1 cubic meter equals 1000 liters.
- `FLOZ` — Imperial system unit of volume (U.S. customary unit).
- `PT` — 1 pint equals 16 fluid ounces (U.S. customary unit).
- `QT` — 1 quart equals 32 fluid ounces (U.S. customary unit).
- `GAL` — 1 gallon equals 128 fluid ounces (U.S. customary unit).
- `MG` — 1000 milligrams equals 1 gram.
- `G` — Metric system unit of weight.
- `KG` — 1 kilogram equals 1000 grams.
- `OZ` — 16 ounces equals 1 pound.
- `LB` — Imperial system unit of weight.
- `MM` — 1000 millimeters equals 1 meter.
- `CM` — 100 centimeters equals 1 meter.
- `M` — Metric system unit of length.
- `IN` — Imperial system unit of length.
- `FT` — 1 foot equals 12 inches.
- `YD` — 1 yard equals 36 inches.
- `M2` — Metric system unit of area.
- `FT2` — Imperial system unit of area.
- `ITEM` — 1 item, a unit of count.
- `UNKNOWN` — The unit of measurement is unknown. Upgrade to the latest version of the API to resolve this unit.
