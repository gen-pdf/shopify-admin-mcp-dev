# Object: `ShopifyqlTableDataColumn`

Represents a column in a ShopifyQL query response.

## Fields

- `dataType`: `ColumnDataType!` — The data type of the column.
- `displayName`: `String!` — The human-readable display name of the column.
- `name`: `String!` — The name of the column.
- `subType`: `ColumnDataType` — The sub type of an array column.
