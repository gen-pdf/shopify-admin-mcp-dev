# Input Object: `CheckoutBrandingInput`

The input fields used to upsert the checkout branding settings.

## Input Fields

- `designSystem`: `CheckoutBrandingDesignSystemInput` — The design system allows you to set values that represent specific attributes
- `customizations`: `CheckoutBrandingCustomizationsInput` — The customizations that apply to specific components or areas of the user interface.
