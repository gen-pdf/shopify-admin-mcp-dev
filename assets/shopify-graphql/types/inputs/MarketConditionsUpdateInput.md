# Input Object: `MarketConditionsUpdateInput`

The input fields required to update a market condition.

## Input Fields

- `conditionsToAdd`: `MarketConditionsInput` — The conditions to update to the market condition.
- `conditionsToDelete`: `MarketConditionsInput` — The conditions to delete from the market condition.
