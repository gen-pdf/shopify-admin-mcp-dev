# Enum: `OrderActionType`

The possible order action types for a
[sales agreement](https://shopify.dev/api/admin-graphql/latest/interfaces/salesagreement).

## Values

- `ORDER` — An order with a purchase or charge.
- `ORDER_EDIT` — An edit to the order.
- `REFUND` — A refund on the order.
- `RETURN` — A return on the order.
- `UNKNOWN` — An unknown agreement action. Represents new actions that may be added in future versions.
