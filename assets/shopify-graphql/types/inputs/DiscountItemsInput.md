# Input Object: `DiscountItemsInput`

The input fields for the items attached to a discount. You can specify the discount items by product ID or collection ID.

## Input Fields

- `products`: `DiscountProductsInput` — The
- `collections`: `DiscountCollectionsInput` — The collections that are attached to a discount.
- `all`: `Boolean` — Whether all items should be selected for the discount. Not supported for Buy X get Y discounts.
