# Object: `CollectionRuleSet`

The set of rules that are used to determine which products are included in the collection.

## Fields

- `appliedDisjunctively`: `Boolean!` — Whether products must match any or all of the rules to be included in the collection.
- `rules`: `[CollectionRule!]!` — The rules used to assign products to the collection.
