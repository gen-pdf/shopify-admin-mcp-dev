# Input Object: `SellingPlanInventoryPolicyInput`

The input fields required to create or update an inventory policy.

## Input Fields

- `reserve`: `SellingPlanReserve` — When to reserve inventory for the order. The value must be ON_FULFILLMENT or ON_SALE.
