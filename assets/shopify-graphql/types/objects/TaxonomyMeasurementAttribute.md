# Object: `TaxonomyMeasurementAttribute`

A Shopify product taxonomy measurement attribute.

**Implements:** `Node`

## Fields

- `id`: `ID!` — The unique ID of the TaxonomyAttribute.
- `name`: `String!` — The name of the product taxonomy attribute. For example, Color.
- `options`: `[Attribute!]!` — The product taxonomy attribute options.
