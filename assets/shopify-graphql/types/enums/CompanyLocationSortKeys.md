# Enum: `CompanyLocationSortKeys`

The set of valid sort keys for the CompanyLocation query.

## Values

- `COMPANY_AND_LOCATION_NAME` — Sort by the `company_and_location_name` value.
- `COMPANY_ID` — Sort by the `company_id` value.
- `CREATED_AT` — Sort by the `created_at` value.
- `ID` — Sort by the `id` value.
- `NAME` — Sort by the `name` value.
- `RELEVANCE` — Sort by relevance to the search terms when the `query` parameter is specified on the connection.
- `UPDATED_AT` — Sort by the `updated_at` value.
