# Enum: `ProductVariantsBulkDeleteUserErrorCode`

Possible error codes that can be returned by `ProductVariantsBulkDeleteUserError`.

## Values

- `PRODUCT_DOES_NOT_EXIST` — Product does not exist.
- `PRODUCT_SUSPENDED` — Product is suspended.
- `CANNOT_DELETE_LAST_VARIANT` — Cannot delete default variant.
- `AT_LEAST_ONE_VARIANT_DOES_NOT_BELONG_TO_THE_PRODUCT` — The variant does not exist.
- `UNSUPPORTED_COMBINED_LISTING_PARENT_OPERATION` — Operation is not supported for a combined listing parent product.
