# Enum: `CustomerPredictedSpendTier`

The valid tiers for the predicted spend of a customer with a shop.

## Values

- `HIGH` — The customer's spending is predicted to be in the top spending range for the shop in the following year.
- `MEDIUM` — The customer's spending is predicted to be in the normal spending range for the shop in the following year.
- `LOW` — The customer's spending is predicted to be zero, or in the lowest spending range for the shop in the following year.
