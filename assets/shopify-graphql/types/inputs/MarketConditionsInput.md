# Input Object: `MarketConditionsInput`

The input fields required to create or update the market conditions.

## Input Fields

- `companyLocationsCondition`: `MarketConditionsCompanyLocationsInput` — The company locations to include in the market conditions.
- `locationsCondition`: `MarketConditionsLocationsInput` — The locations to include in the market conditions.
- `regionsCondition`: `MarketConditionsRegionsInput` — The regions to include in the market conditions.
