# Enum: `ShopBranding`

Possible branding of a shop.
Branding can be used to define the look of a shop including its styling and logo in the Shopify Admin.

## Values

- `SHOPIFY_GOLD` — Shop has Shopify Gold branding.
- `SHOPIFY_PLUS` — Shop has Shopify Plus branding.
- `ROGERS` — Shop has Rogers branding.
- `SHOPIFY` — Shop has Shopify branding.
