# Input Object: `UTMInput`

Specifies the
[Urchin Traffic Module (UTM) parameters](https://en.wikipedia.org/wiki/UTM_parameters)
that are associated with a related marketing campaign.

## Input Fields

- `campaign`: `String!` — The name of the UTM campaign.
- `source`: `String!` — The name of the website or application where the referral link exists.
- `medium`: `String!` — The UTM campaign medium.
