# Input Object: `RefundShippingInput`

The input fields for the shipping cost to refund.

## Input Fields

- `shippingRefundAmount`: `MoneyInput` — The input fields required to refund shipping cost, in the presentment currency of the order.
- `fullRefund`: `Boolean` = `false` — Whether to refund the full shipping amount.
