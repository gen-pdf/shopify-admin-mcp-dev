# Enum: `ConsentPolicyErrorCode`

Possible error codes that can be returned by `ConsentPolicyError`.

## Values

- `COUNTRY_CODE_REQUIRED` — Country code is required.
- `REGION_CODE_REQUIRED` — Region code is required for countries with existing regional policies.
- `REGION_CODE_MUST_MATCH_COUNTRY_CODE` — Region code must match the country code.
- `SHOPIFY_COOKIE_BANNER_NOT_DISABLED` — Shopify's cookie banner must be disabled.
- `UNSUPORTED_CONSENT_POLICY` — Unsupported consent policy.
