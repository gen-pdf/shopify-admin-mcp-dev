# Input Object: `OrderIdentifierInput`

The input fields for identifying a order.

## Input Fields

- `id`: `ID` — The ID of the order.
- `customId`: `UniqueMetafieldValueInput` — The [custom ID](https://shopify.dev/docs/apps/build/custom-data/metafields/working-with-custom-ids) of the order.
