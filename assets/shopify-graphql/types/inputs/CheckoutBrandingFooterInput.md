# Input Object: `CheckoutBrandingFooterInput`

The input fields when mutating the checkout footer settings.

## Input Fields

- `position`: `CheckoutBrandingFooterPosition` — The input field for setting the footer position customizations.
- `divided`: `Boolean` — The divided setting.
- `alignment`: `CheckoutBrandingFooterAlignment` — The footer alignment settings. You can set the footer native content alignment to the left, center, or right.
- `content`: `CheckoutBrandingFooterContentInput` — The input field for setting the footer content customizations.
- `colorScheme`: `CheckoutBrandingColorSchemeSelection` — The selected color scheme of the footer container.
- `padding`: `CheckoutBrandingSpacingKeyword` — The padding of the footer container.
