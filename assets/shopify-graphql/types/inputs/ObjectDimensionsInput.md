# Input Object: `ObjectDimensionsInput`

The input fields for dimensions of an object.

## Input Fields

- `length`: `Float!` — The length in `unit`s.
- `width`: `Float!` — The width in `unit`s.
- `height`: `Float!` — The height in `unit`s.
- `unit`: `LengthUnit!` — Unit of measurement for `length`, `width`, and `height`.
