# Input Object: `AppPlanInput`

The pricing model for the app subscription.
The pricing model input can be either `appRecurringPricingDetails` or `appUsagePricingDetails`.

## Input Fields

- `appUsagePricingDetails`: `AppUsagePricingInput` — The pricing details for usage-based billing.
- `appRecurringPricingDetails`: `AppRecurringPricingInput` — The pricing details for recurring billing.
