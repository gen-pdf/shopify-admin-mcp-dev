# Object: `ShopResourceLimits`

Resource limits of a shop.

## Fields

- `locationLimit`: `Int!` — Maximum number of locations allowed.
- `maxProductOptions`: `Int!` — Maximum number of product options allowed.
- `maxProductVariants`: `Int!` — The maximum number of variants allowed per product.
- `redirectLimitReached`: `Boolean!` — Whether the shop has reached the limit of the number of URL redirects it can make for resources.
