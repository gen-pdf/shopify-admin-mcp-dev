# Input Object: `SubscriptionManualDiscountValueInput`

The input fields for the discount value and its distribution.

## Input Fields

- `percentage`: `Int` — The percentage value of the discount. Value must be between 0 - 100.
- `fixedAmount`: `SubscriptionManualDiscountFixedAmountInput` — Fixed amount input in the currency defined by the subscription.
