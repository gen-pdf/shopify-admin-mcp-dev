# Input Object: `OptionUpdateInput`

The input fields for updating a product option.

## Input Fields

- `id`: `ID!` — Specifies the product option to update.
- `name`: `String` — Name of the option.
- `position`: `Int` — Position of the option.
- `linkedMetafield`: `LinkedMetafieldUpdateInput` — Specifies the metafield the option is linked to.
