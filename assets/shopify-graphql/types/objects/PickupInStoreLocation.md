# Object: `PickupInStoreLocation`

A location for in-store pickup.

## Fields

- `code`: `String!` — The code of the pickup location.
- `distanceFromBuyer`: `Distance` — Distance from the buyer to the pickup location.
- `handle`: `String!` — A unique identifier for this pickup location.
- `instructions`: `String!` — Pickup instructions.
- `locationId`: `ID!` — The location ID of the pickup location.
- `source`: `String!` — The source of the pickup location.
- `title`: `String!` — Title of the pickup location.
