# Enum: `CheckoutProfileSortKeys`

The set of valid sort keys for the CheckoutProfile query.

## Values

- `CREATED_AT` — Sort by the `created_at` value.
- `EDITED_AT` — Sort by the `edited_at` value.
- `ID` — Sort by the `id` value.
- `IS_PUBLISHED` — Sort by the `is_published` value.
- `UPDATED_AT` — Sort by the `updated_at` value.
