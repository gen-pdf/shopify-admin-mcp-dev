# Input Object: `SubscriptionBillingAttemptInput`

The input fields required to complete a subscription billing attempt.

## Input Fields

- `idempotencyKey`: `String!` — A unique key generated by the client to avoid duplicate payments. For more information, refer to [Idempotent requests](https://shopify.dev/api/usage/idempotent-requests).
- `originTime`: `DateTime` — The date and time used to calculate fulfillment intervals for a billing attempt that
- `billingCycleSelector`: `SubscriptionBillingCycleSelector` — Select the specific billing cycle to be billed.
- `inventoryPolicy`: `SubscriptionBillingAttemptInventoryPolicy` = `PRODUCT_VARIANT_INVENTORY_POLICY` — The behaviour to follow when creating an order for a product variant
