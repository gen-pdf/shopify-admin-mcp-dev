# Object: `TenderTransactionCreditCardDetails`

Information about the credit card used for this transaction.

## Fields

- `creditCardCompany`: `String` — The name of the company that issued the customer's credit card. Example: `Visa`.
- `creditCardNumber`: `String` — The customer's credit card number, with all digits except the last 4 redacted. Example: `•••• •••• •••• 1234`
