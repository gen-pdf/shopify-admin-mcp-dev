# Input Object: `CheckoutBrandingCornerRadiusVariablesInput`

The input fields used to update the corner radius variables.

## Input Fields

- `small`: `Int` — The value in pixels for small corner radii. It should be greater than zero. Example: 3.
- `base`: `Int` — The value in pixels for base corner radii. It should be greater than zero. Example: 5.
- `large`: `Int` — The value in pixels for large corner radii. It should be greater than zero. Example: 10.
