# Enum: `AppUninstallAppUninstallErrorCode`

Possible error codes that can be returned by `AppUninstallAppUninstallError`.

## Values

- `APP_NOT_FOUND` — The app cannot be found.
- `APP_NOT_INSTALLED` — The app is not installed.
- `USER_PERMISSIONS_INSUFFICIENT` — User does not have sufficient permissions to uninstall this app.
- `APP_UNINSTALL_ERROR` — An error occurred while uninstalling the app.
