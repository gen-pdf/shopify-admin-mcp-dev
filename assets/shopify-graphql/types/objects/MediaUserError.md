# Object: `MediaUserError`

Represents an error that happens during execution of a Media query or mutation.

**Implements:** `DisplayableError`

## Fields

- `code`: `MediaUserErrorCode` — The error code.
- `field`: `[String!]` — The path to the input field that caused the error.
- `message`: `String!` — The error message.
