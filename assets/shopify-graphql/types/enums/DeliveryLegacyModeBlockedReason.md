# Enum: `DeliveryLegacyModeBlockedReason`

Reasons the shop is blocked from converting to full multi-location delivery profiles mode.

## Values

- `MULTI_LOCATION_DISABLED` *(deprecated)* — Multi-Location mode is disabled. The shop can't convert to full multi-location delivery profiles mode.
- `NO_LOCATIONS_FULFILLING_ONLINE_ORDERS` — There are no locations for this store that can fulfill online orders.
