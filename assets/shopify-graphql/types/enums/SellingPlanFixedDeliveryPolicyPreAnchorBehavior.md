# Enum: `SellingPlanFixedDeliveryPolicyPreAnchorBehavior`

The fulfillment or delivery behavior of the first fulfillment when the orderis placed before the anchor.

## Values

- `ASAP` — Orders placed can be fulfilled / delivered immediately. Orders placed inside a cutoff can be fulfilled / delivered at the next anchor.
- `NEXT` — Orders placed can be fulfilled / delivered at the next anchor date.
