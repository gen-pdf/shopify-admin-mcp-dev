# Object: `ReverseDeliveryLabelV2`

The return label file information for a reverse delivery.

## Fields

- `createdAt`: `DateTime!` — The date and time when the reverse delivery label was created.
- `publicFileUrl`: `URL` — A public link that can be used to download the label image.
- `updatedAt`: `DateTime!` — The date and time when the reverse delivery label was updated.
