# Input Object: `ExchangeLineItemAppliedDiscountValueInput`

The input value for an applied discount on a calculated exchange line item.
Can either specify the value as a fixed amount or a percentage.

## Input Fields

- `amount`: `MoneyInput` — The value of the discount as a fixed amount.
- `percentage`: `Float` — The value of the discount as a percentage.
