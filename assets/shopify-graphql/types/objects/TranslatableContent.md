# Object: `TranslatableContent`

Translatable content of a resource's field.

## Fields

- `digest`: `String` — Hash digest representation of the content value.
- `key`: `String!` — The resource field that's being translated.
- `locale`: `String!` — Locale of the content.
- `type`: `LocalizableContentType!` — Type of the translatable content.
- `value`: `String` — Content value.
