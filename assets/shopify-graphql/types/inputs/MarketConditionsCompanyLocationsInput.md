# Input Object: `MarketConditionsCompanyLocationsInput`

The input fields required to create or update a company location market condition.

## Input Fields

- `companyLocationIds`: `[ID!]` — A list of company location IDs to include in the market condition.
- `applicationLevel`: `MarketConditionApplicationType` — A type of market condition (e.g. ALL) to apply.
