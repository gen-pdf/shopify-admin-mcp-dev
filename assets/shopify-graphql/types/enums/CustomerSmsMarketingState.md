# Enum: `CustomerSmsMarketingState`

The valid SMS marketing states for a customer’s phone number.

## Values

- `NOT_SUBSCRIBED` — The customer hasn't subscribed to SMS marketing.
- `PENDING` — The customer is in the process of subscribing to SMS marketing.
- `SUBSCRIBED` — The customer is subscribed to SMS marketing.
- `UNSUBSCRIBED` — The customer isn't currently subscribed to SMS marketing but was previously subscribed.
- `REDACTED` — The customer's personal data is erased. This value is internally-set and read-only.
