# Input Object: `MobilePlatformApplicationUpdateAndroidInput`

The input fields for an Android based mobile platform application.

## Input Fields

- `applicationId`: `String` — Android application ID.
- `sha256CertFingerprints`: `[String!]` — The SHA256 fingerprints of the app’s signing certificate.
- `appLinksEnabled`: `Boolean` — Whether Android App Links are supported by this app.
