# Enum: `DeliveryPromiseProviderUpsertUserErrorCode`

Possible error codes that can be returned by `DeliveryPromiseProviderUpsertUserError`.

## Values

- `NOT_FOUND` — The record with the ID used as the input value couldn't be found.
- `TOO_LONG` — The input value is too long.
- `MUST_BELONG_TO_APP` — The location doesn't belong to the app.
- `INVALID_TIME_ZONE` — The time zone is invalid.
