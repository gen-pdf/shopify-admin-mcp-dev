# Enum: `ColumnDataType`

The data type of a column.

## Values

- `UNSPECIFIED` — Represents an unspecified data type.
- `MONEY` — Represents a monetary value.
- `PERCENT` — Represents a percentage value.
- `INTEGER` — Represents an integer value.
- `FLOAT` — Represents a floating point value.
- `DECIMAL` — Represents a decimal value.
- `STRING` — Represents a string value.
- `BOOLEAN` — Represents a boolean value.
- `TIMESTAMP` — Represents a timestamp value in seconds.
- `MINUTE_TIMESTAMP` — Represents a minute-level timestamp value.
- `HOUR_TIMESTAMP` — Represents a hour-level timestamp value.
- `DAY_TIMESTAMP` — Represents a day-level timestamp value.
- `WEEK_TIMESTAMP` — Represents a week-level timestamp value.
- `MONTH_TIMESTAMP` — Represents a month-level timestamp value.
- `QUARTER_TIMESTAMP` — Represents a quarter-level timestamp value.
- `YEAR_TIMESTAMP` — Represents a year-level timestamp value.
- `DAY_OF_WEEK` — Represents a day of week value.
- `HOUR_OF_DAY` — Represents an hour of day value.
- `IDENTITY` — Represents an identity value.
- `MONTH_OF_YEAR` — Represents a month of year value.
- `WEEK_OF_YEAR` — Represents a week of year value.
- `SECOND_TIMESTAMP` — Represents a second-level timestamp value.
- `ARRAY` — Represents an array of values.
- `MILLISECOND_DURATION` — Represents a duration in milliseconds.
- `SECOND_DURATION` — Represents a duration in seconds.
- `MINUTE_DURATION` — Represents a duration in minutes.
- `HOUR_DURATION` — Represents a duration in hours.
- `DAY_DURATION` — Represents a duration in days.
