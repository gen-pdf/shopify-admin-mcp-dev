# Object: `OrderPaymentCollectionDetails`

The payment collection details for an order that requires additional payment following an edit to the order.

## Fields

- `additionalPaymentCollectionUrl`: `URL` — The URL to use for collecting an additional payment on the order.
- `vaultedPaymentMethods`: `[PaymentMandate!]` — The list of vaulted payment methods for the order with their permissions.
