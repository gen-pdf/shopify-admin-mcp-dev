# Enum: `SellingPlanFulfillmentTrigger`

Describes what triggers fulfillment.

## Values

- `ANCHOR` — Use the anchor values to calculate fulfillment date.
- `ASAP` — As soon as possible.
- `EXACT_TIME` — At an exact time defined by the fulfillment_exact_time field.
- `UNKNOWN` — Unknown. Usually to be determined in the future.
