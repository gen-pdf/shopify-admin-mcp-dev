# Input Object: `MetaobjectFieldDefinitionOperationInput`

The input fields for possible operations for modifying field definitions. Exactly one option is required.

## Input Fields

- `create`: `MetaobjectFieldDefinitionCreateInput` — The input fields for creating a metaobject field definition.
- `update`: `MetaobjectFieldDefinitionUpdateInput` — The input fields for updating a metaobject field definition.
- `delete`: `MetaobjectFieldDefinitionDeleteInput` — The input fields for deleting a metaobject field definition.
