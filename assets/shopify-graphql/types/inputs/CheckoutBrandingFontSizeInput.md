# Input Object: `CheckoutBrandingFontSizeInput`

The input fields used to update the font size.

## Input Fields

- `base`: `Float` — The base font size. Its value should be between 12.0 and 18.0.
- `ratio`: `Float` — The scale ratio used to derive all font sizes such as small and large. Its value should be between 1.0 and 1.4.
