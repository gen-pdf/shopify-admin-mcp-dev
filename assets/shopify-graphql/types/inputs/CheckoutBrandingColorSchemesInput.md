# Input Object: `CheckoutBrandingColorSchemesInput`

The input fields for the color schemes.

## Input Fields

- `scheme1`: `CheckoutBrandingColorSchemeInput` — The primary scheme. By default, it’s used for the main area of the interface.
- `scheme2`: `CheckoutBrandingColorSchemeInput` — The secondary scheme. By default, it’s used for secondary areas, like Checkout’s Order Summary.
- `scheme3`: `CheckoutBrandingColorSchemeInput` — An extra scheme available to customize more surfaces, components or specific states of the user interface.
- `scheme4`: `CheckoutBrandingColorSchemeInput` — An extra scheme available to customize more surfaces, components or specific states of the user interface.
