# Object: `DiscountShareableUrl`

A shareable URL for a discount code.

## Fields

- `targetItemImage`: `Image` — The image URL of the item (product or collection) to which the discount applies.
- `targetType`: `DiscountShareableUrlTargetType!` — The type of page that's associated with the URL.
- `title`: `String!` — The title of the page that's associated with the URL.
- `url`: `URL!` — The URL for the discount code.
