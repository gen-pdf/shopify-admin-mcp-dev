# Input Object: `AppSubscriptionDiscountInput`

The input fields to specify a discount to the recurring pricing portion of a subscription over a number of billing intervals.

## Input Fields

- `value`: `AppSubscriptionDiscountValueInput` — The value to be discounted every billing interval.
- `durationLimitInIntervals`: `Int` — The total number of billing intervals to which the discount will be applied. Must be greater than 0.
