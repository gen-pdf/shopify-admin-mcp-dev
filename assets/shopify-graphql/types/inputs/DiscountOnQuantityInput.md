# Input Object: `DiscountOnQuantityInput`

The input fields for the quantity of items discounted and the discount value.

## Input Fields

- `quantity`: `UnsignedInt64` — The quantity of items that are discounted.
- `effect`: `DiscountEffectInput` — The percentage value of the discount.
