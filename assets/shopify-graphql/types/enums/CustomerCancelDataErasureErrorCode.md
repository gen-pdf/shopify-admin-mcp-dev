# Enum: `CustomerCancelDataErasureErrorCode`

Possible error codes that can be returned by `CustomerCancelDataErasureUserError`.

## Values

- `DOES_NOT_EXIST` — Customer does not exist.
- `FAILED_TO_CANCEL` — Failed to cancel customer data erasure.
- `NOT_BEING_ERASED` — Customer's data is not scheduled for erasure.
- `UNAUTHORIZED_CANCELLATION` — Only the original requester can cancel this data erasure.
