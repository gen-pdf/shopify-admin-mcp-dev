# Input Object: `SubscriptionDeliveryMethodShippingInput`

Specifies shipping delivery method fields.

This input accepts partial input. When a field is not provided,
its prior value is left unchanged.

## Input Fields

- `address`: `MailingAddressInput` — The address to ship to.
- `shippingOption`: `SubscriptionDeliveryMethodShippingOptionInput` — The details of the shipping method to use.
