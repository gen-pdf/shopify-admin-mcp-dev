# Object: `BundlesFeature`

Represents the Bundles feature configuration for the shop.

## Fields

- `eligibleForBundles`: `Boolean!` — Whether a shop is configured properly to sell bundles.
- `ineligibilityReason`: `String` — The reason why a shop is not eligible for bundles.
- `sellsBundles`: `Boolean!` — Whether a shop has any fixed bundle products or has a cartTransform function installed.
