# Enum: `LocalizableContentType`

Specifies the type of the underlying localizable content. This can be used to conditionally render different UI elements such as input fields.

## Values

- `JSON_STRING` — A JSON string.
- `JSON` — A JSON.
- `LINK` — A link.
- `LIST_LINK` — A list of links.
- `LIST_MULTI_LINE_TEXT_FIELD` — A list of multi-line texts.
- `LIST_SINGLE_LINE_TEXT_FIELD` — A list of single-line texts.
- `LIST_URL` — A list of URLs.
- `MULTI_LINE_TEXT_FIELD` — A multi-line text.
- `RICH_TEXT_FIELD` — A rich text.
- `SINGLE_LINE_TEXT_FIELD` — A single-line text.
- `STRING` — A string.
- `URL` — A URL.
- `FILE_REFERENCE` — A file reference.
- `LIST_FILE_REFERENCE` — A list of file references.
- `HTML` — An HTML.
- `URI` — A URI.
- `INLINE_RICH_TEXT` — An inline rich text.
