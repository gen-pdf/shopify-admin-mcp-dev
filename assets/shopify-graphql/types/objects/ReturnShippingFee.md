# Object: `ReturnShippingFee`

A return shipping fee is a fee captured as part of a return to cover the costs of shipping the return.

**Implements:** `Fee`

## Fields

- `amountSet`: `MoneyBag!` — The amount of the return shipping fee, in shop and presentment currencies.
- `id`: `ID!` — The unique ID for the Fee.
