# Enum: `ArticleSortKeys`

The set of valid sort keys for the Article query.

## Values

- `AUTHOR` — Sort by the `author` value.
- `BLOG_TITLE` — Sort by the `blog_title` value.
- `ID` — Sort by the `id` value.
- `PUBLISHED_AT` — Sort by the `published_at` value.
- `TITLE` — Sort by the `title` value.
- `UPDATED_AT` — Sort by the `updated_at` value.
