# Input Object: `CheckoutBrandingColorsInput`

The input fields used to update the color settings for global colors and color schemes.

## Input Fields

- `global`: `CheckoutBrandingColorGlobalInput` — The input to update global colors for customizing the overall look and feel of the user interface.
- `schemes`: `CheckoutBrandingColorSchemesInput` — The input to define color schemes which apply to different areas of the user interface.
