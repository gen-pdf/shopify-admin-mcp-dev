# Enum: `CalculatedShippingLineStagedStatus`

Represents the staged status of a CalculatedShippingLine on a CalculatedOrder.

## Values

- `NONE` — The shipping line has no staged changes associated with it.
- `ADDED` — The shipping line was added as part of the current order edit.
- `REMOVED` — The shipping line was removed as part of the current order edit.
