# Input Object: `MetaobjectAccessInput`

The input fields that set access permissions for the definition's metaobjects.

## Input Fields

- `admin`: `MetaobjectAdminAccessInput` — The access permitted on the Admin API.
- `storefront`: `MetaobjectStorefrontAccess` — The access permitted on the Storefront API.
- `customerAccount`: `MetaobjectCustomerAccountAccess` — The access permitted on the Customer Account API.
