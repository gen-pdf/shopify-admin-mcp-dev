# Input Object: `CountryHarmonizedSystemCodeInput`

The input fields required to specify a harmonized system code.

## Input Fields

- `harmonizedSystemCode`: `String!` — Country specific harmonized system code.
- `countryCode`: `CountryCode` — The ISO 3166-1 alpha-2 country code for the country that issued the specified harmonized system code. Represents global harmonized system code when set to null.
