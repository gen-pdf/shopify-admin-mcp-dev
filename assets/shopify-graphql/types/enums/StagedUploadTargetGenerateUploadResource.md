# Enum: `StagedUploadTargetGenerateUploadResource`

The resource type to receive.

## Values

- `COLLECTION_IMAGE` — An image associated with a collection.
- `FILE` — Represents any file other than HTML.
- `IMAGE` — An image.
- `MODEL_3D` — A Shopify hosted 3d model.
- `PRODUCT_IMAGE` *(deprecated)* — An image that's associated with a product.
- `SHOP_IMAGE` — An image.
- `VIDEO` — A Shopify-hosted video.
- `BULK_MUTATION_VARIABLES` — Represents bulk mutation variables.
- `RETURN_LABEL` — Represents a label associated with a return.
- `URL_REDIRECT_IMPORT` — Represents a redirect CSV file.
- `DISPUTE_FILE_UPLOAD` — Represents a file associated with a dispute.
