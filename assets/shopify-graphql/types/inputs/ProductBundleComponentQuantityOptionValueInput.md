# Input Object: `ProductBundleComponentQuantityOptionValueInput`

The input fields for a single quantity option value related to a component product.

## Input Fields

- `name`: `String!` — The name associated with the option, e.g. one-pack, two-pack.
- `quantity`: `Int!` — How many of the variant will be included for the option value (e.g. two-pack has quantity 2).
