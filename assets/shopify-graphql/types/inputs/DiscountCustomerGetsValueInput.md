# Input Object: `DiscountCustomerGetsValueInput`

The input fields for the quantity of items discounted and the discount value.

## Input Fields

- `discountOnQuantity`: `DiscountOnQuantityInput` — The quantity of the items that are discounted and the discount value.
- `percentage`: `Float` — The percentage value of the discount. Value must be between 0.00 - 1.00.
- `discountAmount`: `DiscountAmountInput` — The value of the discount.
