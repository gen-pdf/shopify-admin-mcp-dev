# Input Object: `MetafieldAccessInput`

The input fields that set access permissions for the definition's metafields.

## Input Fields

- `admin`: `MetafieldAdminAccessInput` — The access permitted on the Admin API.
- `storefront`: `MetafieldStorefrontAccessInput` — The access permitted on the Storefront API.
- `customerAccount`: `MetafieldCustomerAccountAccessInput` — The access permitted on the Customer Account API.
