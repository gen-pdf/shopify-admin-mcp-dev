# Input Object: `MarketConditionsLocationsInput`

The input fields required to create or update a location market condition.

## Input Fields

- `locationIds`: `[ID!]` — A list of location IDs to include in the market condition.
- `applicationLevel`: `MarketConditionApplicationType` — A type of market condition (e.g. ALL) to apply.
