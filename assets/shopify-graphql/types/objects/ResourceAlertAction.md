# Object: `ResourceAlertAction`

An action associated to a resource alert, such as editing variants.

## Fields

- `primary`: `Boolean!` — Whether the action appears as a button or as a link.
- `show`: `String` — Resource for the action to show.
- `title`: `String!` — The text for the button in the alert. For example, _Edit variants_.
- `url`: `URL!` — The target URL that the button links to.
