# Enum: `ProductCollectionSortKeys`

The set of valid sort keys for products belonging to a collection.

## Values

- `BEST_SELLING` — Sort by best selling.
- `COLLECTION_DEFAULT` — Sort by collection default order.
- `CREATED` — Sort by creation time.
- `ID` — Sort by id.
- `MANUAL` — Sort by manual order.
- `PRICE` — Sort by price.
- `RELEVANCE` — Sort by relevance.
- `TITLE` — Sort by title.
