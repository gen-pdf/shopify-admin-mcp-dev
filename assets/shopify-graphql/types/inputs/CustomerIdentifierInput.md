# Input Object: `CustomerIdentifierInput`

The input fields for identifying a customer.

## Input Fields

- `id`: `ID` — The ID of the customer.
- `customId`: `UniqueMetafieldValueInput` — The [custom ID](https://shopify.dev/docs/apps/build/custom-data/metafields/working-with-custom-ids) of the customer.
- `emailAddress`: `String` — The email address of the customer.
- `phoneNumber`: `String` — The phone number of the customer.
