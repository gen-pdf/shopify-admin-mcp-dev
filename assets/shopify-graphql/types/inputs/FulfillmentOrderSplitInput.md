# Input Object: `FulfillmentOrderSplitInput`

The input fields for the split applied to the fulfillment order.

## Input Fields

- `fulfillmentOrderLineItems`: `[FulfillmentOrderLineItemInput!]!` — The fulfillment order line items to be split out.
- `fulfillmentOrderId`: `ID!` — The ID of the fulfillment order to be split.
