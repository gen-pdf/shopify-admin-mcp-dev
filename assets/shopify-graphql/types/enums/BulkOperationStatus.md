# Enum: `BulkOperationStatus`

The valid values for the status of a bulk operation.

## Values

- `CANCELED` — The bulk operation has been canceled.
- `CANCELING` — Cancelation has been initiated on the bulk operation. There may be a short delay from when a cancelation
- `COMPLETED` — The bulk operation has successfully completed.
- `CREATED` — The bulk operation has been created.
- `EXPIRED` — The bulk operation URL has expired.
- `FAILED` — The bulk operation has failed. For information on why the operation failed, use
- `RUNNING` — The bulk operation is runnning.
