# Enum: `CompanySortKeys`

The set of valid sort keys for the Company query.

## Values

- `CREATED_AT` — Sort by the `created_at` value.
- `ID` — Sort by the `id` value.
- `NAME` — Sort by the `name` value.
- `ORDER_COUNT` — Sort by the `order_count` value.
- `SINCE_DATE` — Sort by the `since_date` value.
- `TOTAL_SPENT` — Sort by the `total_spent` value.
- `UPDATED_AT` — Sort by the `updated_at` value.
