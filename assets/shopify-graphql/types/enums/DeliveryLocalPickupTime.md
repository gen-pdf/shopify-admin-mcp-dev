# Enum: `DeliveryLocalPickupTime`

Possible pickup time values that a location enabled for local pickup can have.

## Values

- `ONE_HOUR` — Usually ready in 1 hour.
- `TWO_HOURS` — Usually ready in 2 hours.
- `FOUR_HOURS` — Usually ready in 4 hours.
- `TWENTY_FOUR_HOURS` — Usually ready in 24 hours.
- `TWO_TO_FOUR_DAYS` — Usually ready in 2-4 days.
- `FIVE_OR_MORE_DAYS` — Usually ready in 5+ days.
- `CUSTOM` *(deprecated)* — Custom pickup time. Unrecognized pickup time enum value.
