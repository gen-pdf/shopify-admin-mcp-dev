# Input Object: `MarketingActivityCreateInput`

The input fields required to create a marketing activity. Marketing activity app extensions are deprecated and will be removed in the near future.

## Input Fields

- `marketingActivityExtensionId`: `ID!` — The ID of the marketing activity extension.
- `status`: `MarketingActivityStatus!` — The current state of the marketing activity.
