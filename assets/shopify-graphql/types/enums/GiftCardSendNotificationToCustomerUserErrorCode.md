# Enum: `GiftCardSendNotificationToCustomerUserErrorCode`

Possible error codes that can be returned by `GiftCardSendNotificationToCustomerUserError`.

## Values

- `INVALID` — The input value is invalid.
- `CUSTOMER_NOT_FOUND` — The customer could not be found.
- `GIFT_CARD_NOT_FOUND` — The gift card could not be found.
