# Input Object: `FulfillmentOrderMergeInputMergeIntent`

The input fields for merging fulfillment orders into a single merged fulfillment order.

## Input Fields

- `fulfillmentOrderLineItems`: `[FulfillmentOrderLineItemInput!]` — The fulfillment order line items to be merged.
- `fulfillmentOrderId`: `ID!` — The ID of the fulfillment order to be merged.
