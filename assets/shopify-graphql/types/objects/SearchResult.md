# Object: `SearchResult`

Represents an individual result returned from a search.

## Fields

- `description`: `String` — Returns the search result description text.
- `image`: `Image` — Returns the Image resource presented to accompany a search result.
- `reference`: `Node!` — Returns the resource represented by the search result.
- `title`: `String!` — Returns the resource title.
- `url`: `URL!` — Returns the absolute URL to the resource in the search result.
