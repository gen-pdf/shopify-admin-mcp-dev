# Enum: `SegmentSortKeys`

The set of valid sort keys for the Segment query.

## Values

- `CREATION_DATE` — Sort by the `creation_date` value.
- `ID` — Sort by the `id` value.
- `LAST_EDIT_DATE` — Sort by the `last_edit_date` value.
- `RELEVANCE` — Sort by relevance to the search terms when the `query` parameter is specified on the connection.
