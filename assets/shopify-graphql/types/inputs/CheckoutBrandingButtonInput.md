# Input Object: `CheckoutBrandingButtonInput`

The input fields used to update the buttons customizations.

## Input Fields

- `background`: `CheckoutBrandingBackgroundStyle` — The background style used for buttons.
- `border`: `CheckoutBrandingSimpleBorder` — The border used for buttons.
- `cornerRadius`: `CheckoutBrandingCornerRadius` — The corner radius used for buttons.
- `blockPadding`: `CheckoutBrandingSpacing` — The block padding used for buttons.
- `inlinePadding`: `CheckoutBrandingSpacing` — The inline padding used for buttons.
- `typography`: `CheckoutBrandingTypographyStyleInput` — The typography style used for buttons.
