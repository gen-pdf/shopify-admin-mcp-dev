# Object: `MetaobjectCapabilities`

Provides the capabilities of a metaobject definition.

## Fields

- `onlineStore`: `MetaobjectCapabilitiesOnlineStore` — Indicates whether a metaobject definition can be displayed as a page on the Online Store.
- `publishable`: `MetaobjectCapabilitiesPublishable!` — Indicate whether a metaobject definition is publishable.
- `renderable`: `MetaobjectCapabilitiesRenderable` — Indicate whether a metaobject definition is renderable and exposes SEO data.
- `translatable`: `MetaobjectCapabilitiesTranslatable!` — Indicate whether a metaobject definition is translatable.
