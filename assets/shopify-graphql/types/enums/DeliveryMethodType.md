# Enum: `DeliveryMethodType`

Possible method types that a delivery method can have.

## Values

- `SHIPPING` — The order is shipped.
- `PICK_UP` — The order is picked up by the customer.
- `NONE` — Non-physical items, no delivery needed.
- `RETAIL` — In-store sale, no delivery needed.
- `LOCAL` — The order is delivered using a local delivery service.
- `PICKUP_POINT` — The order is delivered to a pickup point.
