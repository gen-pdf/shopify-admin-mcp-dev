# Object: `SegmentAssociationFilter`

A filter that takes a value that's associated with an object. For example, the `tags` field is associated with the [`Customer`](/api/admin-graphql/latest/objects/Customer) object.

**Implements:** `SegmentFilter`

## Fields

- `localizedName`: `String!` — The localized name of the filter.
- `multiValue`: `Boolean!` — Whether a file can have multiple values for a single customer.
- `queryName`: `String!` — The query name of the filter.
