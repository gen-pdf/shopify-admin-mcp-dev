# Object: `MetafieldCapabilityAdminFilterable`

Information about the admin filterable capability on a metafield definition.

## Fields

- `eligible`: `Boolean!` — Indicates if the definition is eligible to have the capability.
- `enabled`: `Boolean!` — Indicates if the capability is enabled.
- `status`: `MetafieldDefinitionAdminFilterStatus!` — Determines the metafield definition's filter status for use in admin filtering.
