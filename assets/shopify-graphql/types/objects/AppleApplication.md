# Object: `AppleApplication`

The Apple mobile platform application.

## Fields

- `appClipApplicationId`: `String` — The iOS App Clip application ID.
- `appClipsEnabled`: `Boolean!` — Whether iOS App Clips are enabled for this app.
- `appId`: `String` — The iOS App ID.
- `id`: `ID!` — A globally-unique ID.
- `sharedWebCredentialsEnabled`: `Boolean!` — Whether iOS shared web credentials are enabled for this app.
- `universalLinksEnabled`: `Boolean!` — Whether iOS Universal Links are supported by this app.
