# Input Object: `SubscriptionBillingCycleScheduleEditInput`

The input fields for parameters to modify the schedule of a specific billing cycle.

## Input Fields

- `skip`: `Boolean` — Sets the skip status for the billing cycle.
- `billingDate`: `DateTime` — Sets the expected billing date for the billing cycle.
- `reason`: `SubscriptionBillingCycleScheduleEditInputScheduleEditReason!` — The reason for editing.
