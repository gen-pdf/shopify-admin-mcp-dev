# Input Object: `ProductBundleConsolidatedOptionInput`

The input fields for a consolidated option on a componentized product.

## Input Fields

- `optionName`: `String!` — The name of the consolidated option (e.g., 'Size', 'Color').
- `optionSelections`: `[ProductBundleConsolidatedOptionSelectionInput!]!` — The option selections that define how this consolidated option maps to component options.
