# Object: `ReturnDecline`

Additional information about why a merchant declined the customer's return request.

## Fields

- `note`: `String` — The notification message sent to the customer about their declined return request.
- `reason`: `ReturnDeclineReason!` — The reason the customer's return request was declined.
