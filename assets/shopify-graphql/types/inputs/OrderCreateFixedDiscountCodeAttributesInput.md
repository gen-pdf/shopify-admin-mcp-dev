# Input Object: `OrderCreateFixedDiscountCodeAttributesInput`

The input fields for a fixed amount discount code to apply to an order.

## Input Fields

- `code`: `String!` — The discount code that was entered at checkout.
- `amountSet`: `MoneyBagInput` — The amount that's deducted from the order total. When you create an order, this value is the monetary amount to deduct.
