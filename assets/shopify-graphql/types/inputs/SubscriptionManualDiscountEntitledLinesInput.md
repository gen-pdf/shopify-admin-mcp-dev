# Input Object: `SubscriptionManualDiscountEntitledLinesInput`

The input fields for the subscription lines the discount applies on.

## Input Fields

- `all`: `Boolean` — Specify whether the subscription discount will apply on all subscription lines.
- `lines`: `SubscriptionManualDiscountLinesInput` — The ID of the lines to add to or remove from the subscription discount.
