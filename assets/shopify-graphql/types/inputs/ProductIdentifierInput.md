# Input Object: `ProductIdentifierInput`

The input fields for identifying a product.

## Input Fields

- `id`: `ID` — The ID of the product.
- `customId`: `UniqueMetafieldValueInput` — The [custom ID](https://shopify.dev/docs/apps/build/custom-data/metafields/working-with-custom-ids) of the product.
- `handle`: `String` — The handle of the product.
