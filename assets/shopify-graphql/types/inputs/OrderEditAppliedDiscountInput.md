# Input Object: `OrderEditAppliedDiscountInput`

The input fields used to add a discount during an order edit.

## Input Fields

- `description`: `String` — The description of the discount.
- `fixedValue`: `MoneyInput` — The value of the discount as a fixed amount.
- `percentValue`: `Float` — The value of the discount as a percentage.
