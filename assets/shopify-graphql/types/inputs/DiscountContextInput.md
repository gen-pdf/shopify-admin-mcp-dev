# Input Object: `DiscountContextInput`

The input fields for the buyers who can use this discount.

## Input Fields

- `all`: `DiscountBuyerSelection` — All buyers are eligible for this discount.
- `customers`: `DiscountCustomersInput` — The list of customer IDs to add or remove from the list of customers.
- `customerSegments`: `DiscountCustomerSegmentsInput` — The list of customer segment IDs to add or remove from the list of customer segments.
