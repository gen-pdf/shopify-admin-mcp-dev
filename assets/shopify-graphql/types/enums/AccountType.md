# Enum: `AccountType`

Possible account types that a staff member can have.

## Values

- `REGULAR` — The account can access the Shopify admin.
- `RESTRICTED` — The account cannot access the Shopify admin.
- `INVITED` — The user has not yet accepted the invitation to create an account.
- `REQUESTED` — The admin has not yet accepted the request to create a collaborator account.
- `COLLABORATOR` — The account of a partner who collaborates with the merchant.
- `COLLABORATOR_TEAM_MEMBER` — The account of a partner collaborator team member.
- `SAML` — The account can be signed into via a SAML provider.
- `INVITED_STORE_OWNER` — The user has not yet accepted the invitation to become the store owner.
