# Input Object: `CheckoutBrandingOrderSummarySectionInput`

The input fields used to update the order summary sections customizations.

## Input Fields

- `colorScheme`: `CheckoutBrandingColorSchemeSelection` — The selected color scheme for the order summary sections.
- `background`: `CheckoutBrandingBackground` — The background style of the order summary sections.
- `cornerRadius`: `CheckoutBrandingCornerRadius` — The corner radius of the order summary sections.
- `border`: `CheckoutBrandingSimpleBorder` — The border for the order summary sections.
- `borderStyle`: `CheckoutBrandingBorderStyle` — The border style of the order summary sections.
- `borderWidth`: `CheckoutBrandingBorderWidth` — The border width of the order summary sections.
- `shadow`: `CheckoutBrandingShadow` — The shadow of the order summary sections.
- `padding`: `CheckoutBrandingSpacingKeyword` — The padding of the order summary sections.
