# Input Object: `ProductVariantIdentifierInput`

The input fields for identifying a product variant.

## Input Fields

- `id`: `ID` — The ID of the product variant.
- `customId`: `UniqueMetafieldValueInput` — The [custom ID](https://shopify.dev/docs/apps/build/custom-data/metafields/working-with-custom-ids) of the product variant.
