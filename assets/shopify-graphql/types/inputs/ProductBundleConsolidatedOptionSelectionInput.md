# Input Object: `ProductBundleConsolidatedOptionSelectionInput`

The input fields for a consolidated option selection that maps to component options.

## Input Fields

- `optionValue`: `String!` — The value for this consolidated option selection (e.g., 'Small', 'Medium', 'Large').
- `components`: `[ProductBundleConsolidatedOptionComponentInput!]!` — The component mappings that define how this option value maps to specific component options.
