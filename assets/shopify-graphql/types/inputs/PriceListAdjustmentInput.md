# Input Object: `PriceListAdjustmentInput`

The input fields to set a price list adjustment.

## Input Fields

- `value`: `Float!` — The value of the price adjustment as specified by the `type`.
- `type`: `PriceListAdjustmentType!` — The type of price adjustment, such as percentage increase or decrease.
