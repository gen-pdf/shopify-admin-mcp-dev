# Enum: `MetafieldDefinitionPinUserErrorCode`

Possible error codes that can be returned by `MetafieldDefinitionPinUserError`.

## Values

- `NOT_FOUND` — The metafield definition was not found.
- `PINNED_LIMIT_REACHED` — The pinned limit has been reached for owner type.
- `ALREADY_PINNED` — The metafield definition is already pinned.
- `INTERNAL_ERROR` — An internal error occurred.
- `UNSUPPORTED_PINNING` — The metafield definition does not support pinning.
- `DISALLOWED_OWNER_TYPE` — Owner type can't be used in this mutation.
