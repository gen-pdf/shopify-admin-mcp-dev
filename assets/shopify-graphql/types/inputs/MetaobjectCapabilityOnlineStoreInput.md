# Input Object: `MetaobjectCapabilityOnlineStoreInput`

The input fields for enabling and disabling the Online Store capability.

## Input Fields

- `enabled`: `Boolean!` — Indicates whether the capability should be enabled or disabled.
- `data`: `MetaobjectCapabilityDefinitionDataOnlineStoreInput` — The data associated with the Online Store capability.
