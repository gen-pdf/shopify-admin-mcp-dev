# Input Object: `CollectionUnpublishInput`

The input fields for specifying the collection to unpublish and the sales channels to remove it from.

## Input Fields

- `id`: `ID!` — The collection to create or update publications for.
- `collectionPublications`: `[CollectionPublicationInput!]!` — The channels where the collection is published.
