# Input Object: `CheckoutBrandingDividerStyleInput`

The input fields used to update the page, content, main and order summary dividers customizations.

## Input Fields

- `borderStyle`: `CheckoutBrandingBorderStyle` — The border style for the divider.
- `borderWidth`: `CheckoutBrandingBorderWidth` — The border width for the divider.
