# Enum: `TransferSortKeys`

The set of valid sort keys for the Transfer query.

## Values

- `CREATED_AT` — Sort by the `created_at` value.
- `DESTINATION_NAME` — Sort by the `destination_name` value.
- `EXPECTED_SHIPMENT_ARRIVAL` — Sort by the `expected_shipment_arrival` value.
- `ID` — Sort by the `id` value.
- `NAME` — Sort by the `name` value.
- `ORIGIN_NAME` — Sort by the `origin_name` value.
- `SOURCE_NAME` — Sort by the `source_name` value.
- `STATUS` — Sort by the `status` value.
