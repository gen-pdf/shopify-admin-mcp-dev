# Input Object: `ContextualPublicationContext`

The context data that determines the publication status of a product.

## Input Fields

- `country`: `CountryCode` — The country code used to fetch country-specific publication.
- `companyLocationId`: `ID` — The company location ID used to fetch company-specific publication.
- `locationId`: `ID` — The Location ID used to fetch the publication status of a product.
