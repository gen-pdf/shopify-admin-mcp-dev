# Enum: `GiftCardSortKeys`

The set of valid sort keys for the GiftCard query.

## Values

- `AMOUNT_SPENT` — Sort by the `amount_spent` value.
- `BALANCE` — Sort by the `balance` value.
- `CODE` — Sort by the `code` value.
- `CREATED_AT` — Sort by the `created_at` value.
- `CUSTOMER_NAME` — Sort by the `customer_name` value.
- `DISABLED_AT` — Sort by the `disabled_at` value.
- `EXPIRES_ON` — Sort by the `expires_on` value.
- `ID` — Sort by the `id` value.
- `INITIAL_VALUE` — Sort by the `initial_value` value.
- `UPDATED_AT` — Sort by the `updated_at` value.
