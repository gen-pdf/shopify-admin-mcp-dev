# Input Object: `EmailInput`

The input fields for an email.

## Input Fields

- `subject`: `String` — Specifies the email subject.
- `to`: `String` — Specifies the email recipient.
- `from`: `String` — Specifies the email sender.
- `body`: `String` — Specifies the email body.
- `bcc`: `[String!]` — Specifies any bcc recipients for the email.
- `customMessage`: `String` — Specifies a custom message to include in the email.
