# Input Object: `ProductPublishInput`

The input fields for specifying a product to publish and the channels to publish it to.

## Input Fields

- `id`: `ID!` — The product to create or update publications for.
- `productPublications`: `[ProductPublicationInput!]!` — The publication that the product is published to.
