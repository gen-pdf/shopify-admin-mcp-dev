# Input Object: `ProductUnpublishInput`

The input fields for specifying a product to unpublish from a channel and the sales channels to unpublish it from.

## Input Fields

- `id`: `ID!` — The ID of the product to create or update publications for.
- `productPublications`: `[ProductPublicationInput!]!` — The channels to unpublish the product from.
