# Enum: `GiftCardSendNotificationToRecipientUserErrorCode`

Possible error codes that can be returned by `GiftCardSendNotificationToRecipientUserError`.

## Values

- `INVALID` — The input value is invalid.
- `RECIPIENT_NOT_FOUND` — The recipient could not be found.
- `GIFT_CARD_NOT_FOUND` — The gift card could not be found.
