# Object: `PriceListAdjustment`

The type and value of a price list adjustment.

For more information on price lists, refer to
[Support different pricing models](https://shopify.dev/apps/internationalization/product-price-lists).

## Fields

- `type`: `PriceListAdjustmentType!` — The type of price adjustment, such as percentage increase or decrease.
- `value`: `Float!` — The value of price adjustment, where positive numbers reduce the prices and negative numbers
