# Object: `ResourceFeedback`

Represents feedback from apps about a resource, and the steps required to set up the apps on the shop.

## Fields

- `appFeedback`: `[AppFeedback!]!` *(deprecated)* — Feedback from an app about the steps a merchant needs to take to set up the app on their store.
- `details`: `[AppFeedback!]!` — List of AppFeedback detailing issues regarding a resource.
- `summary`: `String!` — Summary of resource feedback pertaining to the resource.
