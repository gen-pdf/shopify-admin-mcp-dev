# Enum: `MetafieldDefinitionUnpinUserErrorCode`

Possible error codes that can be returned by `MetafieldDefinitionUnpinUserError`.

## Values

- `NOT_FOUND` — The metafield definition was not found.
- `NOT_PINNED` — The metafield definition isn't pinned.
- `INTERNAL_ERROR` — An internal error occurred.
- `APP_CONFIG_MANAGED` — Definition is managed by app configuration and cannot be modified through the API.
- `DISALLOWED_OWNER_TYPE` — Owner type can't be used in this mutation.
