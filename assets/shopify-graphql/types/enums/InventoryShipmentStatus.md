# Enum: `InventoryShipmentStatus`

The status of an inventory shipment.

## Values

- `DRAFT` — The inventory shipment has been created but not yet shipped.
- `IN_TRANSIT` — The inventory shipment is currently in transit.
- `PARTIALLY_RECEIVED` — The inventory shipment has been partially received at the destination.
- `RECEIVED` — The inventory shipment has been completely received at the destination.
- `OTHER` — Status not included in the current enumeration set.
