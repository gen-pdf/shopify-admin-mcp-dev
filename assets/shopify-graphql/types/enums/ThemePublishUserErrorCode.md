# Enum: `ThemePublishUserErrorCode`

Possible error codes that can be returned by `ThemePublishUserError`.

## Values

- `NOT_FOUND` — The record with the ID used as the input value couldn't be found.
- `CANNOT_PUBLISH_THEME_DURING_INSTALL` — Theme publishing is not available during install.
- `THEME_PUBLISH_NOT_AVAILABLE_FOR_THEME_LIMITED_PLAN` — Theme publishing is not allowed on this plan.
