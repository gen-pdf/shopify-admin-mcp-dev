# Enum: `ReverseFulfillmentOrderDispositionType`

The final arrangement of an item from a reverse fulfillment order.

## Values

- `RESTOCKED` — An item that was restocked.
- `PROCESSING_REQUIRED` — An item that requires further processing before being restocked or discarded.
- `NOT_RESTOCKED` — An item that wasn't restocked.
- `MISSING` — An item that was expected but absent.
