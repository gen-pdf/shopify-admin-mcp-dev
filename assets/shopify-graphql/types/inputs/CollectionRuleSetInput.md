# Input Object: `CollectionRuleSetInput`

The input fields for a rule set of the collection.

## Input Fields

- `appliedDisjunctively`: `Boolean!` — Whether products must match any or all of the rules to be included in the collection.
- `rules`: `[CollectionRuleInput!]` — The rules used to assign products to the collection.
