# Object: `UrlRedirectImportUserError`

Represents an error that happens during execution of a redirect import mutation.

**Implements:** `DisplayableError`

## Fields

- `code`: `UrlRedirectImportErrorCode` — The error code.
- `field`: `[String!]` — The path to the input field that caused the error.
- `message`: `String!` — The error message.
