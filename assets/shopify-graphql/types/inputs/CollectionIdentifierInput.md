# Input Object: `CollectionIdentifierInput`

The input fields for identifying a collection.

## Input Fields

- `id`: `ID` — The ID of the collection.
- `customId`: `UniqueMetafieldValueInput` — The [custom ID](https://shopify.dev/docs/apps/build/custom-data/metafields/working-with-custom-ids) of the collection.
- `handle`: `String` — The handle of the collection.
