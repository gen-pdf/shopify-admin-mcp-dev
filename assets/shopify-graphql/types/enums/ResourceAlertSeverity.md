# Enum: `ResourceAlertSeverity`

The possible severity levels for a resource alert.

## Values

- `DEFAULT` — Indicates a neutral alert. For example, an accepted dispute.
- `INFO` — Indicates an informative alert. For example, an escalated dispute.
- `WARNING` — Indicates an informative alert. For example, a new dispute.
- `SUCCESS` — Indicates a success alert. For example, a winning a dispute.
- `CRITICAL` — Indicates a critical alert. For example, a blocked app.
- `ERROR` *(deprecated)* —
