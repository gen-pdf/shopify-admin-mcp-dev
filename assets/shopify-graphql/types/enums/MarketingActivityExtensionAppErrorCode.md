# Enum: `MarketingActivityExtensionAppErrorCode`

The error code resulted from the marketing activity extension integration.

## Values

- `NOT_ONBOARDED_ERROR` — The shop/user must be onboarded to use the app.
- `VALIDATION_ERROR` — The app has returned validation errors.
- `API_ERROR` — The app is either not responding or returning unexpected data.
- `PLATFORM_ERROR` — The app has returned an error when invoking the platform.
- `INSTALL_REQUIRED_ERROR` — The app needs to be installed.
