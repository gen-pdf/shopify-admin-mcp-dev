# Input Object: `OrderCancelRefundMethodInput`

The input fields used to specify the refund method for an order cancellation.

## Input Fields

- `originalPaymentMethodsRefund`: `Boolean` — Whether to refund to the original payment method.
- `storeCreditRefund`: `OrderCancelStoreCreditRefundInput` — Whether to refund to store credit.
