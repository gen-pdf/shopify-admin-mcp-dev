# Enum: `ErrorsWebPixelUserErrorCode`

Possible error codes that can be returned by `ErrorsWebPixelUserError`.

## Values

- `BLANK` — The input value is blank.
- `TAKEN` — The input value is already taken.
- `NOT_FOUND` — The record with the ID used as the input value couldn't be found.
- `INVALID_SETTINGS` — The provided settings does not match the expected settings definition on the app.
- `UNABLE_TO_DELETE` *(deprecated)* — An error occurred and the web pixel couldnt be deleted.
- `NO_EXTENSION` — No extension found.
- `INVALID_CONFIGURATION_JSON` — The provided settings is not a valid JSON.
- `INVALID_SETTINGS_DEFINITION` — The settings definition of the web pixel extension is in an invalid state on the app.
- `UNEXPECTED_ERROR` — An unexpected error occurred.
- `INVALID_RUNTIME_CONTEXT` — The provided runtime context is invalid.
