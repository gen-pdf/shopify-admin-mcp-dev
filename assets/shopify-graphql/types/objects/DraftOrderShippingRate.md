# Object: `DraftOrderShippingRate`

A shipping rate is an additional cost added to the cost of the products that were ordered.

## Fields

- `code`: `String!` — The code of the shipping rate.
- `handle`: `String!` — Unique identifier for this shipping rate.
- `price`: `MoneyV2!` — The cost associated with the shipping rate.
- `source`: `String!` — The source of the shipping rate.
- `title`: `String!` — The name of the shipping rate.
