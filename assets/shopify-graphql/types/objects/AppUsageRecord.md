# Object: `AppUsageRecord`

Store usage for app subscriptions with usage pricing.

**Implements:** `Node`

## Fields

- `createdAt`: `DateTime!` — The date and time when the usage record was created.
- `description`: `String!` — The description of the app usage record.
- `id`: `ID!` — A globally-unique ID.
- `idempotencyKey`: `String` — A unique key generated by the client to avoid duplicate charges.
- `price`: `MoneyV2!` — The price of the usage record.
- `subscriptionLineItem`: `AppSubscriptionLineItem!` — Defines the usage pricing plan the merchant is subscribed to.
