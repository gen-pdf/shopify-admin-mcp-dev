# Input Object: `CustomerMergeOverrideFields`

The input fields to override default customer merge rules.

## Input Fields

- `customerIdOfFirstNameToKeep`: `ID` — The ID of the customer whose first name will be kept.
- `customerIdOfLastNameToKeep`: `ID` — The ID of the customer whose last name will be kept.
- `customerIdOfEmailToKeep`: `ID` — The ID of the customer whose email will be kept.
- `customerIdOfPhoneNumberToKeep`: `ID` — The ID of the customer whose phone number will be kept.
- `customerIdOfDefaultAddressToKeep`: `ID` — The ID of the customer whose default address will be kept.
- `note`: `String` — The note to keep.
- `tags`: `[String!]` — The tags to keep.
