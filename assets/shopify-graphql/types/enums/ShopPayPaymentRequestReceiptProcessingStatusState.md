# Enum: `ShopPayPaymentRequestReceiptProcessingStatusState`

The state of the payment request receipt.

## Values

- `READY` — The payment request is ready and queued to be processed.
- `PROCESSING` — The payment request currently being processed.
- `FAILED` — The payment request processing failed.
- `COMPLETED` — The payment request processing completed successfully.
- `ACTION_REQUIRED` — The payment request requires action from the buyer.
