# Interface: `CommentEventSubject`

The subject line of a comment event.

## Fields

- `hasTimelineComment`: `Boolean!` — Whether the timeline subject has a timeline comment. If true, then a timeline comment exists.
- `id`: `ID!` — A globally-unique ID.
