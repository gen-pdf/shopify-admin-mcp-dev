# Enum: `MandateResourceType`

The type of resource a payment mandate can be used for.

## Values

- `CREDENTIAL_ON_FILE` — A credential stored on file for merchant and customer initiated transactions.
- `CHECKOUT` — A credential stored on file for checkout.
- `DRAFT_ORDER` — A credential stored on file for a Draft Order.
- `ORDER` — A credential stored on file for an Order.
- `SUBSCRIPTIONS` — A credential stored for subscription billing attempts.
