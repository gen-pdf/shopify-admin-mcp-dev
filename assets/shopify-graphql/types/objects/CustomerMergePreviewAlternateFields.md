# Object: `CustomerMergePreviewAlternateFields`

- `defaultAddress`: `MailingAddress` — The default address of a customer.
- `email`: `CustomerEmailAddress` — The email state of a customer.
- `firstName`: `String` — The first name of a customer.
- `lastName`: `String` — The last name of a customer.
- `phoneNumber`: `CustomerPhoneNumber` — The phone number state of a customer.
