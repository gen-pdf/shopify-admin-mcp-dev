# Input Object: `DeliveryPriceConditionInput`

The input fields for a price-based condition of a delivery method definition.

## Input Fields

- `criteria`: `MoneyInput` — The monetary value to compare the price of an order to.
- `operator`: `DeliveryConditionOperator` — The operator to use for comparison.
