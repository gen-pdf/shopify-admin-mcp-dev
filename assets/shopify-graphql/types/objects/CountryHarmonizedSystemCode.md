# Object: `CountryHarmonizedSystemCode`

The country-specific harmonized system code and ISO country code for an inventory item.

## Fields

- `countryCode`: `CountryCode!` — The ISO 3166-1 alpha-2 country code for the country that issued the specified harmonized system code.
- `harmonizedSystemCode`: `String!` — The country-specific harmonized system code. These are usually longer than 6 digits.
