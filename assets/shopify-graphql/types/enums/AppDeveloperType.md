# Enum: `AppDeveloperType`

Possible types of app developer.

## Values

- `SHOPIFY` — Indicates the app developer is Shopify.
- `PARTNER` — Indicates the app developer is a Partner.
- `MERCHANT` — Indicates the app developer works directly for a Merchant.
- `UNKNOWN` — Indicates the app developer is unknown. It is not categorized as any of the other developer types.
