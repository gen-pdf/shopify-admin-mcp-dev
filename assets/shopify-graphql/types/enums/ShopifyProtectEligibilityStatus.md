# Enum: `ShopifyProtectEligibilityStatus`

The status of an order's eligibility for protection against fraudulent chargebacks by Shopify Protect.

## Values

- `PENDING` — The eligibility of the order is pending and has not yet been determined.
- `ELIGIBLE` — The order is eligible for protection against fraudulent chargebacks.
- `NOT_ELIGIBLE` — The order isn't eligible for protection against fraudulent chargebacks.
