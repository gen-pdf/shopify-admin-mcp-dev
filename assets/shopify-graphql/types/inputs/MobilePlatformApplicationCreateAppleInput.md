# Input Object: `MobilePlatformApplicationCreateAppleInput`

The input fields for an Apple based mobile platform application.

## Input Fields

- `appId`: `String` — Apple application ID.
- `universalLinksEnabled`: `Boolean!` — Whether Apple Universal Links are supported by this app.
- `sharedWebCredentialsEnabled`: `Boolean!` — Whether Apple shared web credentials are enabled for this app.
- `appClipsEnabled`: `Boolean` — Whether Apple app clips are enabled for this app.
- `appClipApplicationId`: `String` — The Apple app clip application ID.
