# Enum: `BankingFinanceAppAccess`

The valid types of actions a user should be able to perform in an financial app.

## Values

- `READ_ACCESS` — Read access in the financial app.
- `MOVE_MONEY` — Ability to perform actions that moves money.
- `MONEY_MOVEMENT_RESTRICTED` — Indication that the user has restricted money movement.
- `MONEY_MOVEMENT_BLOCKED_MFA` — Indication that the user has blocked money movement due to MFA disabled.
