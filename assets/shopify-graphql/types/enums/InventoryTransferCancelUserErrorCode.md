# Enum: `InventoryTransferCancelUserErrorCode`

Possible error codes that can be returned by `InventoryTransferCancelUserError`.

## Values

- `TRANSFER_NOT_FOUND` — The transfer was not found.
- `INVALID_TRANSFER_STATUS` — Current transfer status does not support this operation.
- `SHIPMENT_EXISTS` — Shipment already exists for the transfer.
