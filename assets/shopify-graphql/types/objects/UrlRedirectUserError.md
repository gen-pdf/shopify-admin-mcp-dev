# Object: `UrlRedirectUserError`

Represents an error that happens during execution of a redirect mutation.

**Implements:** `DisplayableError`

## Fields

- `code`: `UrlRedirectErrorCode` — The error code.
- `field`: `[String!]` — The path to the input field that caused the error.
- `message`: `String!` — The error message.
