# Object: `DomainLocalization`

The country and language settings assigned to a domain.

## Fields

- `alternateLocales`: `[String!]!` — The ISO codes for the domain’s alternate locales. For example, `["en"]`.
- `country`: `String` — The ISO code for the country assigned to the domain. For example, `"CA"` or "*" for a domain set to "Rest of world".
- `defaultLocale`: `String!` — The ISO code for the domain’s default locale. For example, `"en"`.
