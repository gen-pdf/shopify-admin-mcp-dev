# Enum: `StaffMemberDefaultImage`

Represents the fallback avatar image for a staff member. This is used only if the staff member has no avatar image.

## Values

- `DEFAULT` — Returns a default avatar image for the staff member.
- `TRANSPARENT` — Returns a transparent avatar image for the staff member.
- `NOT_FOUND` — Returns a URL that returns a 404 error if the image is not present.
