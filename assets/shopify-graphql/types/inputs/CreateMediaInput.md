# Input Object: `CreateMediaInput`

The input fields required to create a media object.

## Input Fields

- `originalSource`: `String!` — The original source of the media object. This might be an external URL or a staged upload URL.
- `alt`: `String` — The alt text associated with the media.
- `mediaContentType`: `MediaContentType!` — The media content type.
