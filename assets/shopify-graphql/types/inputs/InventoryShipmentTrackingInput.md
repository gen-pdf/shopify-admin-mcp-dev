# Input Object: `InventoryShipmentTrackingInput`

The input fields for an inventory shipment's tracking information.

## Input Fields

- `trackingNumber`: `String` — The tracking number for the shipment.
- `company`: `String` — The name of the shipping carrier company.
- `trackingUrl`: `URL` — The URL to track the shipment.
- `arrivesAt`: `DateTime` — The estimated date and time that the shipment will arrive.
