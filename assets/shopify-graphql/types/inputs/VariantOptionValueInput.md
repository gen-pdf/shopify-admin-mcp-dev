# Input Object: `VariantOptionValueInput`

The input fields required to create or modify a product variant's option value.

## Input Fields

- `id`: `ID` — Specifies the product option value by ID.
- `name`: `String` — Specifies the product option value by name.
- `linkedMetafieldValue`: `String` — Metafield value associated with an option.
- `optionId`: `ID` — Specifies the product option by ID.
- `optionName`: `String` — Specifies the product option by name.
