The suggested values for a refund to store credit.

**Implements:** `SuggestedRefundMethod`

## Fields

- `expiresAt`: `DateTime` — The suggested expiration date for the store credit.
- `maximumRefundable`: `MoneyBag!` — The maximum available amount to refund in shop and presentment currencies.
