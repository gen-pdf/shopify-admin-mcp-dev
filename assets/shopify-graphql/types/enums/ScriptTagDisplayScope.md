# Enum: `ScriptTagDisplayScope`

The page or pages on the online store where the script should be included.

## Values

- `ALL` *(deprecated)* — Include the script on both the web storefront and the <b>Order status</b> page.
- `ORDER_STATUS` *(deprecated)* — Include the script only on the <b>Order status</b> page.
- `ONLINE_STORE` — Include the script only on the web storefront.
