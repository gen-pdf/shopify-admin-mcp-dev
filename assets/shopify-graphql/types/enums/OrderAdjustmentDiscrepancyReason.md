# Enum: `OrderAdjustmentDiscrepancyReason`

Discrepancy reasons for order adjustments.

## Values

- `RESTOCK` — The discrepancy reason is restocking.
- `DAMAGE` — The discrepancy reason is damage.
- `CUSTOMER` — The discrepancy reason is customer.
- `REFUND_DISCREPANCY` — The discrepancy reason is not one of the predefined reasons.
- `FULL_RETURN_BALANCING_ADJUSTMENT` — The discrepancy reason is balance adjustment.
- `PENDING_REFUND_DISCREPANCY` — The discrepancy reason is pending refund.
