# Enum: `InventoryTransferStatus`

The status of a transfer.

## Values

- `DRAFT` — The inventory transfer has been created but not yet finalized.
- `READY_TO_SHIP` — The inventory transfer has been created, but not yet shipped.
- `IN_PROGRESS` — The inventory transfer is in progress, with a shipment currently underway or received.
- `TRANSFERRED` — The inventory transfer has been completely received at the destination.
- `CANCELED` — The inventory transfer has been canceled.
- `OTHER` — Status not included in the current enumeration set.
