# Input Object: `PaymentTermsInput`

The input fields to create payment terms. Payment terms set the date that payment is due.

## Input Fields

- `paymentTermsTemplateId`: `ID` — Specifies the ID of the payment terms template.
- `paymentSchedules`: `[PaymentScheduleInput!]` — Specifies the payment schedules for the payment terms.
