# Input Object: `CheckoutBrandingLogoInput`

The input fields used to update the logo customizations.

## Input Fields

- `image`: `CheckoutBrandingImageInput` — The logo image (must not be of SVG format).
- `maxWidth`: `Int` — The maximum width of the logo.
- `visibility`: `CheckoutBrandingVisibility` — The visibility of the logo.
