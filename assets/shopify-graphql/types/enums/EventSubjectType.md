# Enum: `EventSubjectType`

The type of the resource that generated the event.

## Values

- `COMPANY_LOCATION` — A CompanyLocation resource generated the event.
- `COMPANY` — A Company resource generated the event.
- `CUSTOMER` — A Customer resource generated the event.
- `DRAFT_ORDER` — A DraftOrder resource generated the event.
- `INVENTORY_TRANSFER` — A InventoryTransfer resource generated the event.
- `COLLECTION` — A Collection resource generated the event.
- `PRODUCT` — A Product resource generated the event.
- `PRODUCT_VARIANT` — A ProductVariant resource generated the event.
- `ARTICLE` — A Article resource generated the event.
- `BLOG` — A Blog resource generated the event.
- `COMMENT` — A Comment resource generated the event.
- `PAGE` — A Page resource generated the event.
- `DISCOUNT_AUTOMATIC_BXGY` — A DiscountAutomaticBxgy resource generated the event.
- `DISCOUNT_AUTOMATIC_NODE` — A DiscountAutomaticNode resource generated the event.
- `DISCOUNT_CODE_NODE` — A DiscountCodeNode resource generated the event.
- `DISCOUNT_NODE` — A DiscountNode resource generated the event.
- `PRICE_RULE` — A PriceRule resource generated the event.
- `ORDER` — A Order resource generated the event.
