# Enum: `MarketingActivityStatusBadgeType`

StatusBadgeType helps to identify the color of the status badge.

## Values

- `DEFAULT` — This status badge has type default.
- `SUCCESS` — This status badge has type success.
- `ATTENTION` — This status badge has type attention.
- `WARNING` — This status badge has type warning.
- `INFO` — This status badge has type info.
- `CRITICAL` — This status badge has type critical.
