# Input Object: `CompanyInput`

The input fields for company attributes when creating or updating a company.

## Input Fields

- `name`: `String` — The name of the company.
- `note`: `String` — A note about the company.
- `externalId`: `String` — A unique externally-supplied ID for the company.
