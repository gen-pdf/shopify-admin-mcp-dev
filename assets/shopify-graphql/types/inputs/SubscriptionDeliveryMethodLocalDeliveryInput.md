# Input Object: `SubscriptionDeliveryMethodLocalDeliveryInput`

The input fields for a local delivery method.

This input accepts partial input. When a field is not provided,
its prior value is left unchanged.

## Input Fields

- `address`: `MailingAddressInput` — The address to deliver to.
- `localDeliveryOption`: `SubscriptionDeliveryMethodLocalDeliveryOptionInput` — The details of the local delivery method to use.
