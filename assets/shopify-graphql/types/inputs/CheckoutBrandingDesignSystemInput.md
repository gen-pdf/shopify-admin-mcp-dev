# Input Object: `CheckoutBrandingDesignSystemInput`

The input fields used to update the design system.

## Input Fields

- `colors`: `CheckoutBrandingColorsInput` — The color settings for global colors and color schemes.
- `typography`: `CheckoutBrandingTypographyInput` — The typography.
- `cornerRadius`: `CheckoutBrandingCornerRadiusVariablesInput` — The corner radius variables.
