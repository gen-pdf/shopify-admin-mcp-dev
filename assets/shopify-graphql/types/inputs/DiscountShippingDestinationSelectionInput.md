# Input Object: `DiscountShippingDestinationSelectionInput`

The input fields for the destinations where the free shipping discount will be applied.

## Input Fields

- `all`: `Boolean` = `false` — Whether the discount code applies to all countries.
- `countries`: `DiscountCountriesInput` — A list of countries where the discount code will apply.
