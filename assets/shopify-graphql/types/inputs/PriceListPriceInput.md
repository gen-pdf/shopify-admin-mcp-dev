# Input Object: `PriceListPriceInput`

The input fields for providing the fields and values to use when creating or updating a fixed price list price.

## Input Fields

- `variantId`: `ID!` — The product variant ID associated with the price list price.
- `price`: `MoneyInput!` — The price of the product variant on this price list.
- `compareAtPrice`: `MoneyInput` — The compare-at price of the product variant on this price list.
