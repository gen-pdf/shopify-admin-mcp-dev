# Input Object: `DeliveryLocationLocalPickupEnableInput`

The input fields for a local pickup setting associated with a location.

## Input Fields

- `locationId`: `ID!` — The ID of the location associated with the location setting.
- `pickupTime`: `DeliveryLocalPickupTime!` — The time of the local pickup.
- `instructions`: `String` — The instructions for the local pickup.
