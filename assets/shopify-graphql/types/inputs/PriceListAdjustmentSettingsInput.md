# Input Object: `PriceListAdjustmentSettingsInput`

The input fields to set a price list's adjustment settings.

## Input Fields

- `compareAtMode`: `PriceListCompareAtMode!` = `ADJUSTED` — Determines how adjustments are applied to compare at prices.
