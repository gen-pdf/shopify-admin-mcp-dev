# Enum: `MetafieldDefinitionAdminFilterStatus`

Possible filter statuses associated with a metafield definition for use in admin filtering.

## Values

- `NOT_FILTERABLE` — The metafield definition cannot be used for admin filtering.
- `IN_PROGRESS` — The metafield definition's metafields are currently being processed for admin filtering.
- `FILTERABLE` — The metafield definition allows admin filtering by matching metafield values.
- `FAILED` — The metafield definition has failed to be enabled for admin filtering.
