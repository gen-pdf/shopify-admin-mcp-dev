# Enum: `UrlRedirectErrorCode`

Possible error codes that can be returned by `UrlRedirectUserError`.

## Values

- `DOES_NOT_EXIST` — Redirect does not exist.
- `CREATE_FAILED` — Redirect could not be created.
- `UPDATE_FAILED` — Redirect could not be updated.
- `DELETE_FAILED` — Redirect could not be deleted.
