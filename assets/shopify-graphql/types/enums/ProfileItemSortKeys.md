# Enum: `ProfileItemSortKeys`

The set of valid sort keys for the ProfileItem query.

## Values

- `CREATED_AT` — Sort by the `created_at` value.
- `ID` — Sort by the `id` value.
- `INVENTORY_TOTAL` — Sort by the `inventory_total` value.
- `PRODUCT_TYPE` — Sort by the `product_type` value.
- `PUBLISHED_AT` — Sort by the `published_at` value.
- `RELEVANCE` — Sort by relevance to the search terms when the `query` parameter is specified on the connection.
- `TITLE` — Sort by the `title` value.
- `UPDATED_AT` — Sort by the `updated_at` value.
- `VENDOR` — Sort by the `vendor` value.
