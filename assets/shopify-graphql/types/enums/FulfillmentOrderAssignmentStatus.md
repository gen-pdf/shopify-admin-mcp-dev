# Enum: `FulfillmentOrderAssignmentStatus`

The assigment status to be used to filter fulfillment orders.

## Values

- `CANCELLATION_REQUESTED` — Fulfillment orders for which the merchant has requested cancellation of
- `FULFILLMENT_REQUESTED` — Fulfillment orders for which the merchant has requested fulfillment.
- `FULFILLMENT_ACCEPTED` — Fulfillment orders for which the merchant's fulfillment request has been accepted.
- `FULFILLMENT_UNSUBMITTED` — Fulfillment orders for which the merchant hasn't yet requested fulfillment.
