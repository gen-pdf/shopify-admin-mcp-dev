# Input Object: `CheckoutBrandingButtonColorRolesInput`

The input fields to set colors for buttons.

## Input Fields

- `background`: `String` — The color of the background.
- `text`: `String` — The color of text.
- `border`: `String` — The color of borders.
- `icon`: `String` — The color of icons.
- `accent`: `String` — The color of accented objects (links and focused state).
- `decorative`: `String` — The decorative color for highlighting specific parts of the user interface.
- `hover`: `CheckoutBrandingColorRolesInput` — The colors of the button on hover.
