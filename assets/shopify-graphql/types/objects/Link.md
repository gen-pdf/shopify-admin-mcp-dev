# Object: `Link`

A link to direct users to.

**Implements:** `HasPublishedTranslations`

## Fields

- `label`: `String!` — A context-sensitive label for the link.
- `translations` `(locale, marketId)`: `[Translation!]!` — The published translations associated with the resource.
- `url`: `URL!` — The URL that the link visits.
