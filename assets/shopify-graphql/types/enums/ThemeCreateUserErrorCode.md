# Enum: `ThemeCreateUserErrorCode`

Possible error codes that can be returned by `ThemeCreateUserError`.

## Values

- `INVALID_ZIP` — Must be a zip file.
- `ZIP_IS_EMPTY` — Zip is empty.
- `ZIP_TOO_LARGE` — May not be used to fetch a file bigger
- `THEME_CREATION_NOT_ALLOWED_FOR_THEME_LIMITED_PLAN` — Theme creation is not allowed for your shop's plan.
- `INVALID_THEME_ROLE_FOR_THEME_CREATION` — Invalid theme role for theme creation.
