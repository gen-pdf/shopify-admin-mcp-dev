# Enum: `ProductVariantsBulkReorderUserErrorCode`

Possible error codes that can be returned by `ProductVariantsBulkReorderUserError`.

## Values

- `PRODUCT_DOES_NOT_EXIST` — Product does not exist.
- `MISSING_VARIANT` — Product variant does not exist.
- `INVALID_POSITION` — Product variant position cannot be zero or negative number.
- `DUPLICATED_VARIANT_ID` — Product variant IDs must be unique.
- `GENERIC_ERROR` — Something went wrong, please try again.
