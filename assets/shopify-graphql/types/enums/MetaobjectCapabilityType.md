# Enum: `MetaobjectCapabilityType`

Metaobject Capabilities types which can be enabled.

## Values

- `PUBLISHABLE` — Allows for a Metaobject to be conditionally publishable.
- `TRANSLATABLE` — Allows for a Metaobject to be translated using the translation api.
- `RENDERABLE` — Allows for a Metaobject to have attributes of a renderable page such as SEO.
- `ONLINE_STORE` — Allows for a Metaobject to be rendered as an Online Store page.
