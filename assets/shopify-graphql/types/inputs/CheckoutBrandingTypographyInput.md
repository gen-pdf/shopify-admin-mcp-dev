# Input Object: `CheckoutBrandingTypographyInput`

The input fields used to update the typography. Refer to the [typography tutorial](https://shopify.dev/docs/apps/checkout/styling/customize-typography)
for more information on how to set these fields.

## Input Fields

- `size`: `CheckoutBrandingFontSizeInput` — The font size.
- `primary`: `CheckoutBrandingFontGroupInput` — A font group used for most components such as text, buttons and form controls.
- `secondary`: `CheckoutBrandingFontGroupInput` — A font group used for heading components by default.
