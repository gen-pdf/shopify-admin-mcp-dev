# Object: `AbandonedCheckoutLineItemComponent`

The list of line item components that belong to a line item.

## Fields

- `id`: `ID!` — A globally-unique ID.
- `image`: `Image` — The variant image associated with the line item component.
- `quantity`: `Int!` — The quantity of the line item component.
- `title`: `String!` — Title of the line item component.
- `variantTitle`: `String` — The name of the variant.
