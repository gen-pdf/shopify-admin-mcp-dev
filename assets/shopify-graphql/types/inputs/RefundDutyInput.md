# Input Object: `RefundDutyInput`

The input fields required to reimburse duties on a refund.

## Input Fields

- `dutyId`: `ID!` — The ID of the duty in the refund.
- `refundType`: `RefundDutyRefundType` — The type of refund for this duty.
