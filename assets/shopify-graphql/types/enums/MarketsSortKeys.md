# Enum: `MarketsSortKeys`

The set of valid sort keys for the Markets query.

## Values

- `CREATED_AT` — Sort by the `created_at` value.
- `ID` — Sort by the `id` value.
- `MARKET_CONDITION_TYPES` — Sort by the `market_condition_types` value.
- `MARKET_TYPE` — Sort by the `market_type` value.
- `NAME` — Sort by the `name` value.
- `STATUS` — Sort by the `status` value.
- `UPDATED_AT` — Sort by the `updated_at` value.
