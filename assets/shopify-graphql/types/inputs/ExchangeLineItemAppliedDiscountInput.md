# Input Object: `ExchangeLineItemAppliedDiscountInput`

The input fields for an applied discount on a calculated exchange line item.

## Input Fields

- `description`: `String` — The description of the discount.
- `value`: `ExchangeLineItemAppliedDiscountValueInput!` — The value of the discount as a fixed amount or a percentage.
