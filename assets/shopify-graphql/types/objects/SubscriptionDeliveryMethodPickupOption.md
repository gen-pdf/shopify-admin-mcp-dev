# Object: `SubscriptionDeliveryMethodPickupOption`

Represents the selected pickup option on a subscription contract.

## Fields

- `code`: `String` — A custom reference to the delivery method for use with automations.
- `description`: `String` — The details displayed to the customer to describe the pickup option.
- `location`: `Location!` — The location where the customer will pick up the merchandise.
- `presentmentTitle`: `String` — The presentment title of the pickup option.
- `title`: `String` — The title of the pickup option.
