# Object: `FinancialKycShopOwner`

Represents the shop owner information for financial KYC purposes.

## Fields

- `email`: `String!` — The email of the shop owner.
- `firstName`: `String` — The first name of the shop owner.
- `id`: `ID!` — A globally-unique ID.
- `lastName`: `String` — The last name of the shop owner.
- `phone`: `String` — The phone number of the shop owner.
