# Input Object: `CheckoutBrandingMainInput`

The input fields used to update the main container customizations.

## Input Fields

- `colorScheme`: `CheckoutBrandingColorSchemeSelection` — The selected color scheme for the main container of the checkout.
- `backgroundImage`: `CheckoutBrandingImageInput` — The background image of the main container (must not be of SVG format).
- `divider`: `CheckoutBrandingContainerDividerInput` — Divider style and visibility on the main container.
- `section`: `CheckoutBrandingMainSectionInput` — The settings for the main sections.
