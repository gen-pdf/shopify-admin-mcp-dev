# Input Object: `LocationIdentifierInput`

The input fields for identifying a location.

## Input Fields

- `id`: `ID` — The ID of the location.
- `customId`: `UniqueMetafieldValueInput` — The [custom ID](https://shopify.dev/docs/apps/build/custom-data/metafields/working-with-custom-ids) of the location.
