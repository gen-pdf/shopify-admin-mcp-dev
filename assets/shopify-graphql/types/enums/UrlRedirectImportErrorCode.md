# Enum: `UrlRedirectImportErrorCode`

Possible error codes that can be returned by `UrlRedirectImportUserError`.

## Values

- `FILE_DOES_NOT_EXIST` *(deprecated)* — CSV file does not exist at given URL.
- `NOT_FOUND` — URL redirect import not found.
- `ALREADY_IMPORTED` — The import has already completed.
- `IN_PROGRESS` — The import is already in progress.
